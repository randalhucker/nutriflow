# NutriFlow User Guide & Manual

> NOTE: Since the UI of NutriFlow is still under heavy construction, these docs contain placeholders for screenshots to be added at a later date.

Welcome to **NutriFlow**, a meal planning, nutrition tracking, and grocery workflow tool designed to help you:
- Plan meals and recipes
- Keep a pantry inventory
- Build shopping carts from meal plans
- Collaborate with a household
- Track daily intake (macros + calories)

> These docs are a living user guide. Some features may be partially implemented depending on the current build.
