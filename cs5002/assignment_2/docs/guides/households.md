# Households (Collaboration)

A Household is a shared space for meal planning and grocery coordination.

Collaboration features:
- A user can join a single household
- Invite other users
- Shared meal plans, pantry, and cart
- Permissions (admin/member/viewer)
- Activity/action log

## Create a Household
1. Go to **Households**
2. Click **Create Household**
3. Name it
4. Save

## Invite members
1. Open household settings
2. Click **Invite**
3. Enter email
4. Send invite

> Screenshot placeholder: `docs/assets/households/invite.png`

## Roles & permissions
Roles:
- **admin**: can manage membership and permissions
- **member**: standard collaborator
- **viewer**: read-only (child-lock)

Per-member toggles:
- can_modify_cart
- can_modify_pantry

## Shared resources
Household-shared objects may include:
- Meal plans
- Pantry
- Carts
- Shared foods/meals/recipes (depending on visibility settings)

## Activity log
The action log can show events like:
- meal_logged
- recipe_added
- item_purchased
- pantry_updated
- cart_modified
- plan_changed
- invite_sent
