# Preferences & Goals

Preferences help NutriFlow personalize recommendations and defaults.

## Preferences you may set
- Diet (omnivore, vegetarian, vegan, keto, etc.)
- Allergens and sensitivities
- Cuisine preferences
- Goals (save money, eat healthier, reduce waste, high protein, bulking, cutting, maintenance)
- Shopping frequency (days)

## Calories: label vs macro-derived (Idea)
Optional feature idea:
- Use reported calories from labels
- Or compute calories from macros (protein/carbs/fat) using 4/4/9 rule

If enabled, this setting impacts:
- Food detail display
- Meal totals
- Intake log totals
